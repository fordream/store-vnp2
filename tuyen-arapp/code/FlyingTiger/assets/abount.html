□ストアにて
ストアでマーカーのステッカーが配布されています。
アプリを起動して「TAKE PHOTO」を選択します。
起動したカメラをマーカーに向けると、オリジナルフォトフレームが登場！
ストア内で撮影を楽しみましょう。

□お部屋にて
フライング タイガー コペンハーゲンのFacebook、または、アプリ紹介ページで配布しているマーカーを印刷・ダウンロードします。
アプリを起動して「TAKE PHOTO」を選択します。
起動したカメラをマーカーに向けると、オリジナルフォトフレームが登場！
フライング タイガー コペンハーゲンの楽しい世界が広がります。

□みんなで撮ろう
フライング タイガー コペンハーゲンのFacebook、または、アプリ紹介ページで配布しているマーカーをスマートフォンに表示します。
別のスマートフォンでアプリを起動して「TAKE PHOTO」を選択します。
マーカーを表示したスマートフォンが写るようにカメラを向けると、オリジナルフォトフレームが登場！
いつでもどこでも、友達や家族と撮影が楽しめます。

※撮影後保存した画像は、静止画になります。
